# README

**https://github.com/guilherme-dambroski/neoway**

O repositório contém:

- Modelos dbt estruturados por camadas  
- Padronizações, regras de negócio e validações  
- Scripts de orquestração e automação  
- Estrutura de testes e documentação  
- Governança dos datasets e chaves  

Todo o desenvolvimento, validação e execução final ocorreram integralmente neste repositório.
